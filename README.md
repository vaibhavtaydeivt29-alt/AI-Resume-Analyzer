# 📄 AI Resume Analyzer

An intelligent Resume Analyzer built using :contentReference[oaicite:0]{index=0} and :contentReference[oaicite:1]{index=1} that helps analyze resumes, detect technical skills, calculate ATS score, and provide improvement suggestions.

## 🚀 Features

- Upload Resume (TXT format)
- Extract text from resume
- Detect technical skills
- Calculate ATS (Applicant Tracking System) score
- Identify missing skills
- Generate improvement suggestions
- Simple and interactive UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ai-resume-analyzer.git
```

Go to project folder:

```bash
cd ai-resume-analyzer
```

Install dependencies:

```bash
pip install streamlit
```

---

## ▶️ Run the Project

```bash
streamlit run app.py
```

---

## 📊 How It Works

1. Upload your resume in TXT format  
2. System extracts resume content  
3. Detects technical skills  
4. Calculates ATS score  
5. Finds missing skills  
6. Provides suggestions for improvement
